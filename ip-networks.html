PRODUCTS : IP networks

Delivery by Type of Traffic, Security Levels, User Categories.

Employing unique IP layer intelligence, with fine-grained policy management, BBT's system enables network managers to implement different levels of security for different users, devices, organizations or user groups. This means multiple public and private networks can operate over the same wireless infrastructure without impacting the integrity or performance of the network as a whole, and without compromising the security or performance of critical or confidential traffic flows.

Many layers of differentiated access can be granted on either a group or individual basis, as each flow is categorized, or encrypted, meaning each gets the appropriate SLA.

Because Blue Bottle Technologies uniquely uses both the radio and IP layers for management, packets are touched in a very minimal way, allowing much more efficient use of bandwidth while mitigating wireless contention.
So while other solutions are controller-based and generally a "star-type" design (meaning all signals have to return to a common control or routing point), BBT is essentially a controllerless architecture using a truly dynamic mesh approach. It provides:

  • A resilient mesh, self-healing network
  • Dynamic indoor and outdoor fixed and mobile communications
  • True multi-cast capabilities
  • Dynamic partitioning and routing
