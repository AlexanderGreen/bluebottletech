Private, Secure Communications Networks

Blue Bottle’s rapid deployment communications network is suited for use in areas where there is little or no infrastructure, with a communications network being established in as little as 15 minutes. By utilising cheap off the shelf GSM handsets end user training is minimised due to the proliferation of devices and their use

Networks can be supplied with ranges of a few hundred metres to tens of kilometres subject to the customer’s requirements.

These private networks can be interconnected to the Public Switched Telephone Network via a number of methods including satellite back haul for use in remote areas.

Multi-Purpose IP Networks

Blue Bottle systems employ unique IP layer intelligence, with fine-grained policy management, enabling network managers to implement different levels of security for different users, devices, organizations or user groups. This means multiple public and private networks can operate over the same wireless infrastructure without impacting the integrity or performance of the network as a whole, and without compromising the security or performance of critical or confidential traffic flows.

Many layers of differentiated access can be granted on either a group or individual basis, as each flow is categorized, or encrypted, meaning each gets the appropriate SLA.

  • Delivery by Type of Traffic - Voice, video, data (small to large files or live streams)
  • Level of Security - Up to military levels of hard encryption on individual streams
  • User Categories - From highly restricted to open, free or for fee paying users

Because BBT uniquely uses both the radio and IP layers for management, packets are touched in a very minimal way, allowing much more efficient use of bandwidth while mitigating wireless contention.

BBT is essentially a controllerless architecture using a truly dynamic mesh approach, providing:

  • A resilient mesh, self-healing network
  • Dynamic indoor and outdoor fixed and mobile communications
  • True multi-cast capabilities
  • Dynamic partitioning and routing
